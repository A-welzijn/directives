# A-Welzijn Directives

v1.0.1

### Hoe het te gebruiken

```javascript
"dependencies": {
	"awelzijn-directives": "latest"
 }
```
```javascript
var app = angular.module('yourApp', [
	'awelzijn.directives'
]);
```

Deze component bevat allerlei algemene directives die worden gebruikt voor de opmaak van pagina's e.d.
